# 导入/导出格式说明

本说明适用于命令行与图形界面的导入/导出功能。

## 导出格式（.txt）
- 每行一个 ID，使用换行分隔。
- 导出内容为数据库中保存的规范化 ID：仅包含字母与数字，不含空格或短横线。
- 文件为普通文本文件，示例：

```
ABCD1234
ABC123
```

## 导入格式（.txt）
- 每行一个 ID。
- 规则：先字母后数字，中间可选一个分隔符（空格或 `-`）。
- 至少包含 1 个字母和 1 个数字。
- 不允许前后空白、不允许额外字符、不允许以数字开头。
- 空行会被视为无效记录；若文件为空或仅空行，将提示“文件为空或只包含空行”。

示例（有效）：
```
ABCD1234
ABC 123
ABC-123
```

示例（无效）：
```
123ABC
ABC-
 ABC123
ABC 123 4
```
