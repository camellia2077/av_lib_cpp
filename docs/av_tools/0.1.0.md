## [v0.1.0] - 2026-04-06

### 新增功能 (Added)
* 新增统一入口命令 `python -m apps.av_tools.run`，支持 `index-formatter`、`move-by-actor`、`fetch-metadata` 子命令。
* 新增版本查询参数 `--version`，输出当前版本 `0.1.0`。
* 新增 `fetch-metadata` 工作流，支持递归扫描视频并通过 API 获取 `title`、`actors`、`cover_url`。
* 新增元数据报告输出，默认写入 `out/av_tools/fetch_metadata` 下的 `result.json`、`result.csv`、`failed.json`、`failed.csv`。

### 技术改进/重构 (Changed/Refactor)
* 重构工具目录结构，将 `tools/index_formatter/` 与 `tools/move_by_actor/` 迁移到 `apps/av_tools/` 下统一维护。
* 重构番号提取链路，在 `fetch-metadata` 中复用 `apps/av_tools/index_formatter/` 标准化规则判定有效番号。
* 重构 API 解析模型，在 `apps/av_tools/move_by_actor/app/models.py` 的 `MovieInfo` 中新增 `cover_url` 字段并兼容多种封面键提取。

### 弃用/删除 (Deprecated/Removed)
* 删除旧入口路径 `tools/index_formatter/` 与 `tools/move_by_actor/`，统一切换到 `apps/av_tools/`。
