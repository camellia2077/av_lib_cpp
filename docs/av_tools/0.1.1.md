## [v0.1.1] - 2026-04-07

### 新增功能 (Added)
* 新增多源元数据抓取链路，支持 `javdb`、`javbus_api`、`r18` 的顺序 fallback，并在报告中输出 `provider` 与 `provider_attempts`。
* 新增独立浏览器 Provider `javbus_browser`，支持通过 Playwright + `bs4` 抓取并解析 `title`、`actors`、`tags`、`cover_url`、`date`、`studio`、`series` 全字段。
* 新增 `fetch-metadata` 的 `--codes-json` 输入方式，可跳过目录扫描直接按 JSON 番号抓取。
* 新增命令行覆盖参数 `--providers`、`--download-mode`、`--provider-timeout`、`--show-browser`，并保持命令行优先级高于 `runtime.toml`。
* 新增 `--download-mode browser-only` 行为，允许仅使用 `javbus_browser` 完成抓取。

### 技术改进/重构 (Changed/Refactor)
* 重构 `providers` 运行时配置模型，扩展 `browser_fallback` 与 `javbus_browser` 配置段并统一由 `apps/av_tools/move_by_actor/runtime.toml` 驱动。
* 重构 Provider 管理器装配逻辑，支持按 `enabled` 顺序动态启用/禁用 Provider 并兼容浏览器模式切换。
* 重构 `fetch-metadata` 进度展示，`[TRY]` 状态改为 `tqdm` 固定行动态刷新，减少终端刷屏。

### 修复 (Fixed)
* 修复 `javbus_browser` 在默认上下文下易进入 `Age Verification` 页面的问题，补充浏览器 `UA/locale` 并增强拦截页识别。
* 修复 `javbus` 搜索时空格变体（如 `IPSD 048`）高频触发 `403` 导致误导性错误日志的问题，过滤高风险 key 变体。
* 修复 `Ctrl+C` 中断不稳定问题，支持首击优雅中断、二次强制退出并保证退出码为 `130`。
