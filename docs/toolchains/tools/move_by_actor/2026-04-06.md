## 2026-04-06

### 新增功能 (Added)
* 新增 `tools/move_by_actor/` 工具链，用于扫描视频文件名中的番号并按演员维度归档搬运。
* 新增 `tools/move_by_actor/run.py` 入口，支持从命令行触发扫描、查询与搬运流程。
* 新增 API 查询链路：通过 `tools/move_by_actor/app/api_client.py` 与 `app/api_runtime.py` 读取外部 API 返回结果并提取演员信息。
* 新增文件搬运能力：根据 API 返回演员列表，将视频移动到对应演员目录（`app/file_ops.py` + `app/service.py`）。
* 新增运行时配置能力：通过 `tools/move_by_actor/runtime.toml` 与 `app/runtime_config.py` 管理 API 与目录参数。

### 技术改进/重构 (Changed/Refactor)
* 重构模块分层为 `app/` 子模块，按职责拆分 `scanner`、`models`、`service`、`file_ops`、`config`、`cli`。
* 重构执行流程为“扫描番号 -> 调用 API -> 解析返回 -> 目标目录决策 -> 搬运文件”，统一由 `app/service.py` 协调。

### 修复 (Fixed)
* 修复命令行运行链路中的配置装配与参数透传，确保 API 返回值可稳定驱动搬运决策。
