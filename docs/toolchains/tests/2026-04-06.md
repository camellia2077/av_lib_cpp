## 2026-04-06

### 新增功能 (Added)
* 新增 C++ 核心测试可执行目标 `avlib_core_tests`，覆盖 `src/core/utils/validator.cpp` 与 `src/core/io/text_file_reader.cpp` 的关键行为。
* 新增测试源文件 `tests/cpp/core_tests.cpp`，提供基础断言与独立可执行测试入口。
* 新增流程测试数据文件 `tests/smoke/test_ids.txt` 与 `tests/smoke/expected_export_ids.txt`，用于 CLI 端到端回归校验。

### 技术改进/重构 (Changed/Refactor)
* 重构测试执行方式为 Python 编排：通过 `python tools/script/run.py test-core` 统一执行 C++ 核心测试（`cmake` + `ctest`）。
* 重构端到端冒烟执行方式：通过 `python tools/script/run.py smoke-cli` 统一执行 CLI 流程测试，并将日志与导出结果落到 `out/smoke/`。

### 修复 (Fixed)
* 修复 Windows 编码导致的 CLI 冒烟读取异常：为子进程输出读取增加 `encoding='utf-8'` 与 `errors='replace'`。
* 修复冒烟输入换行兼容问题：将输入序列统一为 `\n`，避免交互解析异常导致导出步骤失败。
